NMT New Minimalist Dock for XFCE
-----------------------------------

![nmt-plugin](https://i.imgur.com/9wrvuM4.png)
![nmt-plugin](https://i.imgur.com/1oooQUS.png)

A WIP modern taskbar for XFCE

C++ - GTK3 - CSS - DOCK-LIKE - MINIMALISM

1. Build
```
./autogen.sh
make
```

2. Install
```
sudo make install
```

Build dependency :
libxfce4panel-2.0
libxfce4ui-2
gtk-3.0
cairo-1.16
libwnck-3.0
