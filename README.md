Build & Install
-----------------------------------

Docklike is published in the ArchLinux User Repository (AUR) : `yay xfce4-docklike-plugin-git`

On other distros:
```
./autogen.sh
make
sudo make install
```

Building dependency :
libxfce4panel-2.0
libxfce4ui-2
gtk-3.0
cairo-1.16
libwnck-3.0
